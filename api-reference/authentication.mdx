## Base URLs

All base URLs in this reference point to the **dev** environment. The production equivalents are identical, with `-dev` removed from the subdomain.

## Authentication

This API does not yet implement bearer token authentication. All endpoints are currently unprotected - treat request payloads and response data as sensitive and limit access accordingly.

Bearer authentication is planned and will be added in a future release.
